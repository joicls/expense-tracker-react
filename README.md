# Pencatat Pengeluaran

Aplikasi pencatat pengeluaran berbasis React, Vite, Tailwind CSS, Recharts, dan Lucide React.

## Fitur

- Ringkasan pengeluaran bulanan
- Piechart pengeluaran per kategori
- Daftar pengeluaran dengan fitur sortir
- Kategori anggaran yang bisa ditambah, diedit, dan diberi warna
- Penyimpanan lokal menggunakan localStorage

## Menjalankan Proyek

```bash
npm install
npm run dev
```

## Build Produksi

```bash
npm run build
```
