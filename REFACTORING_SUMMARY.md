## Fitur yang Diimplementasikan

### 1. Manajemen Transaksi
- Tambah transaksi baru
- Edit transaksi yang ada
- Hapus transaksi
- Filter transaksi per bulan
- Sorting (terbaru, terlama, terbesar, terkecil, kategori)

### 2. Manajemen Kategori
- Tambah kategori baru dengan warna custom
- Edit nama, budget, dan warna kategori
- Hapus kategori (dengan validasi penggunaan)
- Budget tracking dengan progress bar

### 3. Visualisasi Data
- **Pie Chart**: Distribusi pengeluaran per kategori
- **Line Chart**: Trend pengeluaran 3/6/12 bulan
- **Legend interaktif** dengan warna kategori

### 4. UI/UX
- Dark theme dengan Tailwind CSS
- Responsive design
- Loading states
- Error handling
- Interactive tooltips
